Core Feature Pillars

Resume Parser

Accepts PDF, DOCX, or text inputs and extracts skills, roles, industries, experience years, and achievements using NLP and embeddings.

  • Parsing via pdfminer.six / python-docx
  • spaCy powered named entity recognition
  • OpenAI embeddings for semantic tagging

Job Research & Matching

Connects to LinkedIn Jobs or SerpAPI, summarizes postings, and computes a holistic match percentage. 

match_score = (
    skill_overlap * 0.4 +
    exp_relevance * 0.3 +
    title_similarity * 0.15 +
    location_match * 0.1 +
    tone_alignment * 0.05
)

Resume Optimizer

Scores clarity, structure, keyword relevance, and ATS readiness, delivering visual feedback and suggested improvements.
